# TF2TeamPreview

A minimal TF2 HUD mod that adds RED/BLU team preview buttons to the loadout screen.

## Features

- Adds two buttons ("R" and "B") to the left side of the character model in the loadout screen
- Switch between RED and BLU team skins to preview how your loadout looks on either team
- No other HUD changes - keeps the stock TF2 interface

## Installation

### Step 1: Download

1. Click the green **Code** button at the top of this page
2. Click **Download ZIP**
3. Save the file to your Downloads folder

### Step 2: Extract

1. Open your Downloads folder
2. Right-click the downloaded ZIP file
3. Select **Extract Here**

### Step 3: Install

1. Open the extracted folder
2. Copy the `TfPreview` folder
3. Navigate to your TF2 custom folder:
   - **Linux:** `~/.steam/steam/steamapps/common/Team Fortress 2/tf/custom/`
   - **Windows:** `C:\Program Files (x86)\Steam\steamapps\common\Team Fortress 2\tf\custom\`
   - **Mac:** `~/Library/Application Support/Steam/steamapps/common/Team Fortress 2/tf/custom/`
4. Paste the `TfPreview` folder inside the `custom` folder

### Step 4: Launch

1. If TF2 is running, close it completely
2. Start TF2
3. Open your loadout screen to see the new buttons

## Usage

In the loadout screen, click:
- **R** - Preview RED team skin
- **B** - Preview BLU team skin
